# Post cover images

Place your actual post cover images here. They are served from the app root (and copied to `dist/` on build).

**Expected filenames for local posts:**

- `youth-labor-market.png` — Youth Employment by Gender and Occupation
- `top-universities-gpa.png` — Top 10 Universities by Average GPA
- `startup-ecosystem-2026.png` — Uzbekistan Startup Ecosystem 2026
- `women-in-higher-ed.png` — Women Surpass Men in Higher Education Enrollment

**Fallback (when a post has no cover):** `placeholder.png`

You can use `.png`, `.jpg`, or `.webp`; update the paths in `src/data.ts` if you use different extensions.
